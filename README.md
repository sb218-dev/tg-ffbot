# 🍔 TG FastFood — Система предзаказов для Telegram

Полноценное MVP-решение для автоматизации предзаказов в сети фастфуда. Включает в себя Telegram Mini App для клиентов, панель управления для кухни и бэкенд на Node.js.

## 🚀 Основные возможности

*   **Клиентская часть (Mini App):**
    *   Интерактивная карта (Яндекс Карты) для удобного визуального выбора заведения.
    *   Автоматическая авторизация по Telegram ID и авто-переход к последней выбранной точке.
    *   Динамическое меню в формате сетки (grid) с категориями, поиском и фотографиями блюд.
    *   Система добавок и соусов (addons) со стилизованными карточками и динамическим расчетом цен.
    *   Валидация времени заказа (минимум за 20 минут, максимум на 2 суток).
    *   Слияние корзины и экрана оформления.
    *   Интерактивный полноэкранный чек текущего активного заказа с функцией быстрой отмены.
    *   Современный UI: плавные анимации переходов, эффекты нажатия кнопок, Haptic Feedback.
*   **Панель кухни (`kitchen.html`):**
    *   Защита входа (Basic Auth).
    *   Управление статусами заказов (Новый -> Готов / Отменен).
    *   Перенос времени готовности заказа (редактирование с мгновенным авто-уведомлением клиента).
    *   Управление стоп-листами (мгновенное скрытие товаров).
    *   Редактор меню и локаций (добавление, изменение цен, привязка добавок).
    *   **Smart-Архивация**: Прошедшие (вчерашние) заказы автоматически сжимаются (gzip) и переносятся в структуру папок по дням и точкам, освобождая память SQL-базы.
    *   **Интеграции агрегаторов**: Поддержка заказов из Яндекс Еда, Купер, QuickResto с цветовой градацией и фильтрами по источникам/точкам.
*   **Бэкенд & Бот:**
    *   Уведомления кухни в специальный чат.
    *   Уведомления клиентов о готовности/отмене через бота.
    *   **Smart Delay:** Автоматическая отправка извинений клиенту, если заказ не готов вовремя.

## 🛠 Установка и запуск

### 1. Требования
*   Node.js v16+
*   Токен бота от @BotFather

### 2. Клонирование и установка

```bash
npm install
```

### 3. Настройка окружения

Создайте файл `config.js` в корне проекта и добавьте ваши секретные данные:

```javascript
module.exports = {
    TOKEN: 'ВАШ_ТОКЕН_БОТА',
    WEBAPP_URL: 'https://ваш-домен.ru',
    KITCHEN_CHAT_ID: 'ID_ЧАТА_КУХНИ'
};
```

### 4. Запуск

Продакшн-режим:
```bash
npm start
```

## 🤖 Команды бота

- `/start` — Запуск бота, приветствие и появление кнопки для открытия Web App (выбор заведения и меню)

## 📂 Структура проекта

```text
tg-fastfood/
├── server.js           # Точка входа, запуск сервера и фоновые задачи
├── database.js         # Подключение к SQLite, создание таблиц
├── config.js           # Настройки (токены, ID чатов) - НЕ ПУШИТЬ В GIT
├── routes/             # API модули (маршруты)
│   ├── admin.js        # Запросы панели управления и кухни
│   ├── client.js       # Запросы клиентского приложения
│   └── integrations.js # Webhook'и для интеграций (Yandex, Kuper, QuickResto)
├── public/             # Фронтенд (статика)
│   ├── index.html      # Приложение для клиентов (Mini App)
│   ├── app.js          # Логика клиента (JS)
│   ├── style.css       # Стили клиента (CSS)
│   ├── kitchen.html    # Интерфейс кухни и админки
│   ├── test.html       # Панель для тестирования генерации заказов
│   └── image/          # Папка для статических изображений
└── package.json
```
