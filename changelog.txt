- UI/UX: Эффект прозрачности при скролле категорий теперь работает симметрично с обеих сторон (слева и справа).
- UI/UX: Кнопка "Мой заказ" перенесена с экрана локаций в меню выбора товаров.

- Админ-панель: Добавлен фильтр заказов по точкам.
- Админ-панель: Внедрена цветовая градация заказов по источникам (Telegram/Yandex/Kuper/Quickresto) и их логическая сортировка.
- Интеграции: Создан роутер integrations.js (базовые webhook endpoint'ы).
- Интеграции: Создан файл integrations.example.js, а боевой integrations.js добавлен в .gitignore для защиты ключей авторизации.
- Тестирование: Добавлена скрытая страница /test.html для удобной генерации тестовых заказов с разных агрегаторов в один клик.
